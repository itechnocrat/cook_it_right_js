- структуру языка,  
- расширения языка,  
- сопутствующие инструменты программирования  
и  
- последовательность (очерёдность) изучения и освоения всего этого.

Смотри файл `the_structure_of_JavaScript_and_the_sequence_of_its_study.md`

В этом файле состоит из двух частей:  
- описание структуры языка или другими словами, состава языка  
- описание последовательности изучения языка и окружающей его экосистемы
